# dragon-adaptive-demo
Adaptive HUD prototype for real-time motion, attention, and context-aware display on smart glasses.
D
RAGON Adaptive is a working iOS + SDK smart glasses demo for an adaptive cue model.

The system takes motion, attention, and context as inputs, then decides what information should be shown, hidden, or surfaced in real time. Instead of displaying a constant dashboard, DRAGON adapts the HUD to the user’s state: low-motion moments can show richer context, while high-motion moments strip the display down to only the most important cue.

The demo runs on an iPhone, connects to an SDK glasses over BLE, mirrors the HUD on the phone, and renders live cues directly on the glasses.
